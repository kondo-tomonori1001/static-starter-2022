## 開発用コマンド
``` bash
npm run dev
```
## 公開用ファイル生成コマンド
``` bash
npm run build
```
## ESlint及びPrettierでの構文チェック用コマンド
``` bash
npm run lint
```

## ESLint及びPrettierのフォーマット実行コマンド
``` bash
npm run format
```

## Pre-commitによる構文チェック
huskyを用いてコミット時にESlint及びPrettierでの構文チェックを実行します。
もし、エラーがあった場合にはコミット出来ません。

## Node.jsパッケージの管理
### 最新になっていないパッケージを確認
``` bash
npm outdate
```
